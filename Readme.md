# Prerequisites

Install Python
Install pip (LINK: https://pip.pypa.io/en/stable/installation/#get-pip-py)

## Conda Important Commands

    - conda create -n python_basics --> this command is used to create a new environment
    - conda env list --> this command is used to list all the environments
    - conda activate python_basics --> this command is used to activate the environment
    - conda deactivate --> this command is used to deactivate the environment
    - conda list --> this command is used to list all the packages in an environment
    - <python bin file path> -m pip install <package_name> --> Install packages on a specific python environment or main environment
